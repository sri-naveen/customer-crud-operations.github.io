CRUD Application for CUSTOMER

Project Description:
This project showcases a basic CRUD (Create, Read, Update, Delete) application for managing customer data in a database. The application provides a user-friendly interface to interact with the customer table, allowing users to add new customers, retrieve customer information, update details, and delete customer records.

## Table of Contents:
- Introduction
- Features
- Prerequisites
- Installation
- Usage
- Login Credentials

# Introduction
The web application serves as a practical example of CRUD operations on a customer table. It provides a seamless user experience for managing customer data efficiently.

# Features
- Create: Add new customers to the database.
- Read: Retrieve customer information.
- Update: Modify customer details.
- Delete: Remove customers from the database.

#Prerequisites
Ensure you have the following tools and libraries installed before running the application:
- IDE: Eclipse (Java, JSP)
- JDK: Java Development Kit 1.8 or later
- Server: Apache Tomcat 9.0
- Database Connector: MySQL Connector Java 8.2.0

# Installation
- Clone the repository to your local machine.
- Open the project in Eclipse.
- Set up the MySQL database and configure the connection details.

# Usage
-The application is designed to be user-friendly.
-Follow the instructions provided in the interface for adding, updating, and deleting customer records.

# Login Credentials
- As there is no authentication implemented, the application accepts any combination of username and password.

# Running the Application
- Navigate to src/main/webapp/login.jsp.
- Deploy the application on Apache Tomcat.
- Access the application through a web browser.
